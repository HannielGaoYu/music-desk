# 音乐桌面端应用

### 1.运行方法

* 打开 vs code 或者其他开发工具
* 调出终端输入npm run dev后回车

### 2.打包方法

* 打开 vs code 或者其他开发工具
* 调出终端输入npm run build后回车
* 找到对应文件夹
